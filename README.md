# 🌊 小心地滑！救出寶可夢

<img width="1909" height="986" alt="image" src="https://github.com/user-attachments/assets/4534f69d-53b1-47d1-b739-2710aea35cae" />

一款在滑溜地面上搬運寶可夢、建立基地、累積財富的休閒多人遊戲。支援**單人模式**與**四人連線模式**。

---

## 🎮 遊戲玩法

長長的道路上，各種稀有度的寶可夢散落各區。你的目標：

1. **走到寶可夢旁邊** 按 `E` 抓起（最多同時攜帶 3 隻）
2. **帶回基地** 按 `E` 存入 → 每隻持續產生收入
3. **或靠近商店** 按 `F` 賣出獲得一次性收益
4. 累積金幣 → 解鎖中階、高階、頂階區域的超稀有寶可夢
5. 全隊合計達到 **$1,000T** 即可觸發勝利！

> ⚠️ **注意海嘯！** 每隔約 45 秒發生，捲到時損失 1/4 金幣並掉落手持寶可夢。

## ✨ 主要特色

### 🌊 震撼海嘯特效
- 海嘯來臨時出現巨型 **蒼龍惡鬼** CSS 鬼臉動畫（憤怒眉、發光眼、七支獠牙、滴水）
- 巨浪高達 18 格、24 個飛濺粒子、6 個電漿光暈、藍色邊緣發光

### 🎁 禮包系統
- 每隔一段時間，野外區域隨機出現禮包（廣播通知）
- 走近即可撿起，到商店按 `G` 開啟
- 獎勵：金幣（25–40% of 目前金幣）、速度升級、或免費彩票

### 🎫 彩票抽獎
- 每 **3 分鐘**開獎一次，每輪最多購買 **3 張**
- 費用：目前金幣的 10%
- 獎金池由所有玩家收入自動累積，中獎者獲得全額

### ⭐ 黃金時刻
- 伺服器不定期觸發：全隊收入 **×2.5**，持續 45 秒
- 把握時機快速存入高等寶可夢！

### 🏆 閃光・區域之王
- 每個區域有一隻特別的「閃光」寶可夢，出現時全場廣播
- 收益為該區最高等級的 115%，顏色為閃耀金色

## 🗺️ 區域系統

地圖共 **12 個區域（Zone 1–12）**，分為 4 個階段：

| 階段 | 區域 | 寶可夢等級 | 解鎖費用 |
|------|------|-----------|---------|
| 低階 | Zone 1–3 | Lv 1–3 | 免費 |
| 中階 | Zone 4–6 | Lv 4–6 | $5,000,000 |
| 高階 | Zone 7–9 | Lv 7–20 | $5,000,000,000 |
| 頂階 | Zone 10–12 | Lv 18–30 | $5,000,000,000,000 |

每個區域兩側有巨型裝飾寶可夢飄浮，作為視覺地標。

## 🕹️ 操作說明

| 按鍵 | 動作 |
|------|------|
| `W A S D` 或方向鍵 | 移動 |

## 🌊 海嘯機制

- 每隔 **~45 秒**觸發一次（伺服器同步，多人同時面對）
- 觸發前 **8 秒**出現警告 + 計時條
- **波浪範圍內（非基地區域）被捲到**：
  - 強制傳送回基地
  - 手持寶可夢全部掉落
  - 損失 **1/4 金幣**
- **安全區**：躲在基地或凹槽（每個區域末端的下凹平台）

- 禮包搶先撿走、稀有寶可夢搶先抓，多人競合機制

    ├── main.js             # 單人遊戲主程式
    ├── main_multi.js       # 多人遊戲主程式

    ├── base.js             # 基地邏輯（存入、收入、座位）
    ├── economy.js          # 金幣、升級、彩票系統
    ├── road.js             # 地圖與區域建構
    ├── tsunamiMechanic.js  # 海嘯機制（3D 波浪 + CSS 鬼臉特效）
    ├── remote_player.js    # 多人遠端玩家渲染
    ├── network.js          # WebSocket 網路管理
    ├── shop.js             # 商店 UI（速度/磁力升級、禮包、彩票）

    ├── audio.js            # 音效管理
    └── touch_controls.js   # 手機觸控支援

## ⚙️ 環境需求

- **Python 3.8+**
- **websockets** 套件（多人模式需要）
- 現代瀏覽器（Chrome / Edge / Firefox，支援 WebGL）

```bash
pip install websockets
```

---

## 🚀 啟動方式

### 單人模式

```bash
python server.py
```

瀏覽器開啟 → `http://localhost:7778`

---

### 多人模式（同一台電腦）

```bash
python server_multi.py
```

瀏覽器開啟 → `http://localhost:7778/multi.html`

最多 **4 位玩家**同時連線，各自選擇隊伍顏色（紅／藍／綠／橙）。

---

### 多人模式（區域網路 LAN）

1. **主機**執行伺服器：
   ```bash
   python server_multi.py
   ```

2. 查詢主機的區域網路 IP：
   ```bash
   # Windows
   ipconfig
   # Mac / Linux
   ifconfig
   ```

3. **其他玩家**在瀏覽器輸入：
   ```
   http://主機IP:7778/multi.html?host=主機IP
   ```
   例如：`http://192.168.1.100:7778/multi.html?host=192.168.1.100`
