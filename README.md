# 🌊 小心地滑！救出寶可夢

一款在滑溜地面上搬運寶可夢、建立基地、累積財富的休閒遊戲。支援**單人模式**與**四人連線模式**。

![遊戲截圖](https://raw.githubusercontent.com/kohsusu/pokemon-slip-game/main/preview.png)

---

## 🎮 遊戲玩法

在長長的道路上，各種寶可夢散落在地面。你的目標是：

1. **走到寶可夢旁邊** 按 `E` 抓起牠
2. **帶回基地** 按 `E` 存入（每隻持續產生收入）
3. **或直接販售** 按 `F` 賣出獲得一次性收益
4. 累積足夠金錢 → 解鎖中階、高階區域的稀有寶可夢
5. **⚠️ 注意海嘯！** 每隔 30～90 秒隨機發生，若被捲走會損失 1/4 的錢並掉落手持寶可夢

## ⚙️ 環境需求

- **Python 3.8+**
- **websockets** 套件（多人模式需要）
- 現代瀏覽器（Chrome / Edge / Firefox）

```bash
pip install websockets
```

## 🚀 啟動方式

### 單人模式

```bash
python server.py
```

瀏覽器開啟 → `http://localhost:7778`

---

### 多人模式（同一台電腦）

```bash
python server_multi.py
```

瀏覽器開啟 → `http://localhost:7778/multi.html`

最多 **4 位玩家**同時連線，各自選擇隊伍顏色（紅／藍／綠／橙）。

---

### 多人模式（區域網路 LAN）

1. **主機**執行伺服器：
   ```bash
   python server_multi.py
   ```

2. 查詢主機的區域網路 IP：
   ```bash
   # Windows
   ipconfig
   # Mac / Linux
   ifconfig
   ```

3. **其他玩家**在瀏覽器輸入：
   ```
   http://主機IP:7778/multi.html?host=主機IP
   ```
   例如：`http://192.168.1.100:7778/multi.html?host=192.168.1.100`

## 🕹️ 操作說明

| 按鍵 | 動作 |
|------|------|
| `W A S D` 或方向鍵 | 移動 |
| `空白鍵` | 切換衝刺（再按恢復正常速度）|
| `E` | 抓取寶可夢 / 存入基地 |
| `F` | 賣出手持寶可夢（立即獲得金錢）|
| `B` | 開啟商店（需靠近商店）|
| `1`–`8` | 發送表情符號（多人模式）|

### 商店升級

靠近商店按 `B` 可購買：
- **速度升級**：移動速度提升（最高 5 級）
- **抓地力升級**：減少在濕滑地面打滑（最高 5 級）

### 區域解鎖

| 區域 | 解鎖費用 |
|------|---------|
| 低階區（Zone 1–3）| 免費 |
| 中階區（Zone 4–6）| $5,000,000 |
| 高階區（Zone 7–9）| $5,000,000,000 |

## 🌊 海嘯機制

- 每隔 **30～90 秒**隨機觸發一次
- 觸發前 **8 秒**會出現警告
- 若海嘯通過時你還在寶可夢區域（非基地、非凹槽）：
  - 被強制傳送回基地
  - 手持寶可夢全部掉落
  - **損失 1/4 的金錢**
- 躲在**凹槽**（每個區域末端的下凹平台）可以安全躲避

## 🤝 多人模式特色

- 所有玩家共享同一個寶可夢世界（伺服器權威）
- 某玩家抓走的寶可夢，其他人的畫面上會消失
- 可以看到其他玩家**搬運中的寶可夢**圖示
- 可以看到其他玩家**基地上存放的寶可夢**
- 海嘯同步發生（所有人同時面對）
- 存入或賣出寶可夢後，系統立即在同區域補充新的
